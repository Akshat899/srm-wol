The Four Rules.

1.No Plagiarism

Plagiarism is a crime globally and one should stay away from it for their own safety.

2.Avoid Comments.

Comments should only be written when it's not possible to clearly communicate the intent of a variable or method by using a name.The comment that serves an actual purpose that could not be easily expressed in the code.

3.Know your code.

It seems obvious but it isn't. always know what your code is supposed to do before you start writing it.

4.Test your code.

Always test your codes before you ship it and unnecessarily waste time.
